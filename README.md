# Full-Stack-blogisovellus

*Tämä työ on Full Stack -harjoitystyönä toteutettu blogisovellus. Valmiilla sovelluksella rekisteröitynyt ja kirjautunut käyttäjä voi kirjoittaa blogikirjoituksia omalle tililleen, hakea ja lukea muiden blogikirjoituksia sekä arvostella niitä. Kirjautumaton käyttäjä voi ainoastaan hakea ja lukea muiden blogikirjoituksia. Työ on koodattu javascriptillä ja ulkoasun toteuttamisessa on käytetty html:ää. Ulkoasukirjastona käytetään Bootstrapia.* <br />
